# Student-Records
This is our final semester project
Student Record Management System (Qt & Excel

This is a Student Record Management System developed for the First Semester Final Project.
The project is built using Qt and uses Excel files to store and manage student data. It calculates GPA for each student and organizes student information efficiently.

How to Use
Open the project in Qt Creator.
Run the application.
Input student details (Name, F-Name, District, CAST, GPA).

GPA will be calculated automatically.

Save or load records from Excel files.
